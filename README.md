## ‚ú® Features
- **Real-Time Dashboard:** View total assets and active transactions instantly.
- **Inventory Management:** Add new equipment with pricing and quantity tracking.
- **Borrowing Workflow:** "Borrow" and "Return" cycle with automatic quantity updates.
- **Visual Analytics:** Interactive Doughnut Chart visualizing "Released" vs "Available" items.
- **Secure Authentication:** Server-side password validation (Node.js).
- **Report Export:** One-click CSV export for admin reporting.
- **Mobile Responsive:** Fully functional on desktop and mobile devices.
